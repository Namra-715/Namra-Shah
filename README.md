# Personal Portfolio Website

A minimalist personal portfolio website built with HTML, CSS, and JavaScript. Features a dark/light mode toggle, smooth scrolling navigation, and a contact form.

## ✨ Features

- **Responsive Design** - Works on all devices
- **Dark/Light Mode** - Toggle between themes
- **Smooth Scrolling** - Animated navigation between sections
- **Contact Form** - Functional contact form with email integration
- **Typewriter Font** - JetBrains Mono for a developer aesthetic
- **Minimalist UI** - Clean, professional design

## 🚀 Getting Started

### Prerequisites
- Git
- GitHub account
- Vercel account (free)

### Local Development
1. Clone the repository
2. Open `index.html` in your browser
3. Make changes and test locally

### Deployment
1. Push to GitHub
2. Connect to Vercel
3. Add environment variables
4. Deploy automatically

## 📧 Contact Form Setup

The contact form uses Resend for email delivery:

1. Sign up at [resend.com](https://resend.com)
2. Get your API key
3. Add environment variables in Vercel:
   - `YOUR_EMAIL`: Your email address
   - `RESEND_API_KEY`: Your Resend API key

## 🎨 Customization

- **Colors**: Modify CSS variables in `styles.css`
- **Content**: Update text in `index.html`
- **Styling**: Adjust CSS classes and properties
- **Functionality**: Modify JavaScript in `script.js`
